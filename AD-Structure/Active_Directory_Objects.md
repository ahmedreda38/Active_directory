| <span style='color: #9fef00;'>Object Name</span>                           | <span style='color: #9fef00;'>**Classification**</span> | <span style='color: #9fef00;'>**Security Principal? (Has SID)**</span> | <span style='color: #9fef00;'>**Definition & Attributes**</span>                             | <span style='color: #9fef00;'>**Pentest Relevance (Why we care)**</span>                                                                 |
| -------------------------------------------------------------------------- | ------------------------------------------------------- | ---------------------------------------------------------------------- | -------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------- |
| <span style='color: #9fef00;'>**Users**</span>                             | Leaf                                                    | <span style='color: green;'>**YES**</span>                             | Standard accounts for people.<br>_(Attrs: Display Name, Email, LastLogon, PasswordLastSet)_  | **Primary Target.** Access allows domain enumeration. Low-priv users often have read access to 90% of the domain.                        |
| <span style='color: #9fef00;'>**Computers**</span>                         | Leaf                                                    | <span style='color: green;'>**YES**</span>                             | Represents a machine (workstation/server).<br>_(Attrs: **OS**, **Version**, DNS Name)_       | **High Value.** Compromising a computer (SYSTEM account) gives you valid domain credentials (machine account) to query AD.               |
| <span style='color: #9fef00;'>**Groups**</span>                            | Container                                               | <span style='color: green;'>**YES**</span>                             | Collection of users, computers, or other groups. Used for permission management.             | **Critical.** We look for **"Nested Groups"** (groups inside groups) to find unintended paths to Domain Admin. _Tool: BloodHound._       |
| <span style='color: #9fef00;'>**OUs (Organizational Units)**</span>        | Container                                               | <span style='color: red;'>**NO**</span>                                | Administrative containers for applying **Group Policy (GPO)** and delegating admin tasks.    | **Persistance & ACLs.** Attackers check OUs to see who has "Write" access (delegated rights) to reset passwords for everyone in that OU. |
| <span style='color: #9fef00;'>**Domain**</span>                            | Container                                               | <span style='color: red;'>**NO**</span>                                | The core logical boundary of the network. Contains specific policies (like Password Policy). | **The Scope.** Defines the boundary of trust. The Password Policy here dictates our brute-force strategy.                                |
| <span style='color: #9fef00;'>**Domain Controller**</span>                 | Server (Key Role)                                       | <span style='color: green;'>**YES**</span>                             | The "Brains." Handles auth, stores the database (`NTDS.dit`), enforces policy.               | **The Crown Jewels.** Access here equals total domain dominance (Golden Ticket attacks).                                                 |
| <span style='color: #9fef00;'>**Foreign Security Principals (FSP)**</span> | Leaf (Placeholder)                                      | <span style='color: red;'>**NO**</span> (Holds external SID)           | Represents an object from a _trusted external forest_ (e.g., a user from a partner company). | **Trust Attacks.** If we see many FSPs, we investigate **Domain Trusts** to see if we can pivot to/from an external partner network.     |
| <span style='color: #9fef00;'>**Contacts**</span>                          | Leaf                                                    | <span style='color: red;'>**NO**</span>                                | Info only (external vendors/clients). No login rights.                                       | **Social Engineering.** Good for building phishing lists, but cannot be used for authentication.                                         |
| <span style='color: #9fef00;'>**Printers**</span>                          | Leaf                                                    | <span style='color: red;'>**NO**</span>                                | Points to a network printer resource.                                                        | **Recon.** Rarely a direct target, but useful for mapping network segments.                                                              |
| <span style='color: #9fef00;'>**Shared Folders**</span>                    | Leaf                                                    | <span style='color: red;'>**NO**</span>                                | Pointers to file shares.                                                                     | **Lateral Movement.** We scan these for sensitive data (passwords in scripts, config files).                                             |
| <span style='color: #9fef00;'>**sites**</span>                             | Container (Logical)                                     | <span style='color: red;'>**NO**</span>                                | Physical grouping of subnets for replication efficiency.                                     | **Network Mapping.** Tells us the physical layout of the company (e.g., "NY-Office" vs "London-Office").                                 |
| <span style='color: #9fef00;'>**Built-in**</span>                          | Container                                               | <span style='color: red;'>**NO**</span>                                | Holds default groups created at domain setup.                                                | **Enumeration.** Contains standard groups like "Print Operators" or "Backup Operators" which often have abuseable privileges.            |
